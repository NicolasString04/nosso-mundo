# Nosso Mundo

![GitHub deploied](https://img.shields.io/badge/Deploy-Vercel-black?style=for-the-badge&logo=vercel&logoColor=white)
![Database](https://img.shields.io/badge/Database-Firebase-orange?style=for-the-badge&logo=firebase&logoColor=white)

O **Nosso Mundo** é um sistema romântico interativo desenvolvido com o objetivo de criar uma experiência emocional, moderna e totalmente personalizada. A aplicação funciona como um álbum digital exclusivo para guardar, organizar e eternizar memórias especiais através de fotos, músicas e relatos, proporcionando uma imersão única e nostálgica.

---

##  Objetivo do Projeto

Transformar memórias em uma experiência digital viva, oferecendo um espaço seguro, bonito e dinâmico para registrar a história de um relacionamento.

---

## Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e ferramentas:

* **Front-end:** HTML5, CSS3 e JavaScript (ES6+)
* **Back-end / Lógica:** Python
* **Banco de Dados:** Firebase Firestore (Armazenamento de dados e relatos)
* **Autenticação:** Firebase Authentication (Segurança e acesso restrito)
* **Hospedagem / Deploy:** Vercel

---

##  Funcionalidades Principais

- **Álbum de Memórias:** Registro e exibição de fotos marcantes com relatos dedicados.
- **Contador de Tempo:** Exibição interativa do tempo exato juntos.
- **Integração Musical:** Trilhas sonoras para acompanhar a navegação e o clima da aplicação.
- **Sistema de Autenticação:** Login seguro para garantir que o espaço seja estritamente privado.
- **Sincronização em Tempo Real:** Dados salvos na nuvem via Firebase.
